# Домашнее задание 3

## Задание

Написать скрипт, который:

1. Создаёт папку `/opt/app`
2. Создаёт файл `/opt/app/log.txt`
3. Каждые **17 секунд** записывает в файл `/opt/app/log.txt` случайную строку длиной **до 20 символов**  
   (каждая новая строка — с новой строки)

---

## Скрипт

#!/bin/bash

# 1. Создать папку /opt/app
mkdir -p /opt/app

# 2. Создать файл /opt/app/log.txt
touch /opt/app/log.txt

# 3. Бесконечный цикл записи случайных строк каждые 17 секунд
while true
do  
  # Создание переменной, которая возвращает случайное число с длиной строки от 1 до 20
  length=$((RANDOM % 20 + 1))

  # Отбор случайной строки с буквами (A-Z, a-z) и цифрами (0-9)
  random_string=$(cat /dev/urandom | tr -dc 'A-Za-z0-9' | head -c $length)

  # Вывод строки и добавление её в конец файла log.txt с новой строки
  echo "$random_string" >> /opt/app/log.txt

  # Приостановка выполнения на 17 секунд
  sleep 17
done
